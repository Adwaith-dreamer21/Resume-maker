# Resume Maker App

A modern and user-friendly Resume Builder application that helps users create professional, ATS-friendly resumes with ease. This project is designed to simplify resume creation while providing smart suggestions and real-time preview.

---

## Features

- Create professional resumes in minutes
- Real-time resume preview
- Multiple resume templates
- ATS-friendly formatting
- Smart suggestions for improving resume content
- Add, edit, and delete sections (Education, Skills, Projects, Experience)
- Export resume as PDF
- Clean and responsive user interface
